SITE LEDGER — how to open
-------------------------
1. Save "Site Ledger.html" somewhere on your computer (e.g. Desktop).
2. Double-click it. It opens in your web browser (Chrome, Edge, etc.).
   There is nothing to install and it does not connect to the internet.
3. Start on the "Import timesheets" tab and choose your Excel file(s).

A sample file (sample-timesheet.xlsx) is included so you can try it out.

Note: your data stays in your own browser on your own computer.
Use Backup > Export now and then to save a copy you can keep.
